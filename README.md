### home-server/

iPhoneからClaude Code（CLI版）を使うための自宅サーバー構築ガイド

- [server-setup-log.md](home-server/server-setup-log.md) - セットアップ手順
- [quick-reference.md](home-server/quick-reference.md) - よく使うコマンド集

**構成:**
```
[iPhone] ──Tailscale VPN──→ [自宅Linuxサーバー] ──→ [Claude Code]
```

**使用ツール:**
- Ubuntu Server
- Tailscale（VPN）
- Syncthing（ファイル同期）
- Claude Code
- Blink Shell（iPhoneアプリ）
